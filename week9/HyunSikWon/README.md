# 쿼드 압축 후 개수 세기

# 여행 경로
간단한 DFS 문제인 줄 알았는데 중복되는 경로가 포함될 수 있다는 점과 알파벳 순을 지켜야 한다는 점 때문에 조건을 검사하기가 까다로웠습니다. 실패하는 테스트 케이스를 살펴본 뒤 문제를 해결할 수 있었습니다. 

테스트 케이스: [["ICN", "A"], ["ICN", "A"], ["A","ICN"]], [["ICN", "A"], ["ICN","B"], ["B", "ICN"]]
