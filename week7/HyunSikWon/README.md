# 7주차 문제
## 가장 큰 수

compare 함수를 구현해서 문제를 풀었습니다. 처음엔 조금 헤매다가 단순하게 어떤 값이 앞에 있을 때 더 큰 수를 만들 수 있는지 검사하도록 구현했습니다.

## 멀쩡한 사각형

수학적이 문제라 어려웠습니다. 최대공약수를 이용한 문제입니다. 사실 아직도 이해가 잘 안 가서 좀 더 살펴봐야 할 것 같아요.

## 광고 삽입

문자열로 어떻게 해보려다가 결국 해답을 보고 문제를 풀었습니다. 시간을 초 단위로 바꿨고, 시작점 종료점에 각각 +1, -1 한 뒤 배열을 순회하며 해당 구간의 누적 사용자를 구하는 방법을 사용했습니다. 시간을 초 단위로 바꾸는 방식은 간단한 아이디언데 지금까지 비슷한 문제를 풀면서는 떠올리지 못했었습니다. 후에 비슷한 문제를 만났을 때 사용할 수 있을 것 같습니다. 구간 내의 사용자 수를 구하는 방식도 처음 접해서, 어려운 문제를 풀어야 다양한 방식을 접할 수 있구나 생각이 들기도 했습니다. 해법만 잘 알면 쉽게 풀 수 있는 문제였습니다.
